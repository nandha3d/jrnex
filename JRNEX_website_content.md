## 🎨 Design Direction

- **Aesthetic**: Professional, modern, minimal — no heavy gradients or colorful distractions
- **Color Palette**: Dark navy / charcoal for primary, white for backgrounds, single accent (e.g., electric blue or steel teal)
- **Typography**: Clean sans-serif, strong hierarchy (e.g., Sora or DM Sans for headings, Inter for body)
- **Icons**: Lucide or Phosphor icons — line style, consistent weight
- **Imagery**: Real drone/tech photography, not stock illustrations
- **Animations**: Subtle scroll-reveal, fade-in on card entry, hover state transitions only
- **UI Feel**: B2B SaaS meets aerospace — data-driven, trustworthy, premium

```typescript
const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'about', component: AboutComponent },

  { path: 'solutions/mapping', component: MappingComponent },
  { path: 'solutions/inspection', component: InspectionComponent },
  { path: 'solutions/mining', component: MiningComponent },
  { path: 'education', component: EducationComponent },
  { path: 'education/skill-development', component: SkillDevelopmentComponent },
  { path: 'rpto', component: RptoComponent },
  { path: 'blog', component: BlogComponent },
  { path: 'blog/:slug', component: BlogPostComponent },
  { path: 'contact', component: ContactComponent },
  { path: '**', redirectTo: '' }

```
